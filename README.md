# Siit Frontend Challenge

## Guidelines

- To start the app:
  - run `yarn install`
  - run `cd src && yarn install && cd ..`
  - run `yarn run start-server`
  - run `yarn run start-client`

## Challenge

- [x] Level 1: Fetching our employees
- [x] Level 2: Fetching our services
